# 市场规模 (TAM-SAM-SOM)

## 简介
TAM-SAM-SOM是评估市场规模和市场机会的三层框架，帮助产品团队理解潜在市场规模、可服务市场规模和可获得市场规模。它是商业计划、融资和战略决策的重要基础。

## 核心概念

### 三层市场规模

#### TAM (Total Addressable Market) 总可触达市场
- **定义**: 产品/服务的总体潜在市场需求
- **假设**: 100%市场占有率的理想情况
- **用途**: 展示市场天花板，吸引投资
- **计算方法**: 自上而下、自下而上、价值理论法

#### SAM (Serviceable Addressable Market) 可服务市场
- **定义**: 基于产品能力和资源可触达的市场部分
- **限制因素**: 地理、客户类型、渠道能力
- **用途**: 规划中期业务目标
- **关系**: SAM ≤ TAM

#### SOM (Serviceable Obtainable Market) 可获得市场
- **定义**: 短期内实际可获取的市场份额
- **限制因素**: 竞争、品牌、资源、时间
- **用途**: 制定现实可行的营收目标
- **关系**: SOM ≤ SAM ≤ TAM

### 计算方法

#### 自上而下法
```
TAM = 总市场金额 × 目标细分占比
例: 全球软件市场$500B × SaaS占比20% = $100B
```

#### 自下而上法
```
TAM = 客户数 × 客单价
例: 1000万中小企业 × $1000/年 = $10B
```

#### 价值理论法
```
TAM = 为用户创造的总价值
例: 节省人工时间 × 人力成本 × 潜在用户数
```

## 实现方式 / 应用方法

### 市场规模分析步骤
1. **定义产品/服务**: 明确核心价值和目标客户
2. **确定边界**: 直接相关市场和相邻市场
3. **数据收集**: 行业报告、统计数据、专家访谈
4. **TAM计算**: 使用多种方法交叉验证
5. **SAM细化**: 应用地理、客户、渠道限制
6. **SOM确定**: 考虑竞争、资源和增长阶段
7. **敏感性分析**: 测试关键假设的变化影响

### 数据来源
- **行业报告**: Gartner、IDC、Forrester、艾瑞
- **政府统计**: 统计局、行业协会数据
- **上市公司财报**: 行业龙头财报分析
- **研究论文**: 学术机构市场研究
- **专家访谈**: 行业专家和客户访谈

### 常见错误
- 过度乐观估计TAM
- 忽略市场增长或萎缩趋势
- 混淆相关市场和可触达市场
- 使用不准确的数据源
- 未考虑竞争对SOM的影响

## 应用场景

- **融资路演**: 向投资人展示市场机会
- **战略规划**: 确定市场进入优先级
- **营收预测**: 制定可实现的销售目标
- **投资评估**: 评估项目或市场的吸引力
- **产品定位**: 选择合适的市场细分

## 面试要点

1. **TAM-SAM-SOM的关系?** TAM≥SAM≥SOM，逐层聚焦
2. **哪个数据最重要?** SOM最实际，TAM用于融资
3. **如何提高SOM?** 品牌建设、渠道扩展、产品差异化
4. **市场规模的验证?** 多数据源交叉，敏感性分析
5. **早期产品如何估算?** 类似市场类比，MVP验证
