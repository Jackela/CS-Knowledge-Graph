## 概述

**市场分析 (Market Analysis)** 是产品决策的基石。它帮助团队理解**市场机会**、**竞争格局**和**用户需求**，从而做出基于数据的决策，而非主观猜测。

```
┌─────────────────────────────────────────────────────────────────┐
│                    市场分析在产品周期中的作用                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   发现机会 ──> 定义产品 ──> 开发产品 ──> 上市推广 ──> 持续优化  │
│      │            │            │            │            │      │
│      ▼            ▼            ▼            ▼            ▼      │
│   ┌─────┐     ┌─────┐     ┌─────┐     ┌─────┐     ┌─────┐     │
│   │市场 │     │需求 │     │用户 │     │定位 │     │反馈 │     │
│   │规模 │     │验证 │     │测试 │     │信息 │     │分析 │     │
│   └─────┘     └─────┘     └─────┘     └─────┘     └─────┘     │
│      │            │            │            │            │      │
│      └────────────┴────────────┴────────────┴────────────┘      │
│                                                                 │
│   核心问题:                                                      │
│   • 市场有多大? (TAM/SAM/SOM)                                   │
│   • 谁在竞争? (竞品分析)                                         │
│   • 用户是谁? (用户研究)                                         │
│   • 如何取胜? (竞争策略)                                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 市场分析方法

### SWOT分析

**SWOT分析**评估企业的优势、劣势、机会和威胁，是战略规划的基础工具。

```
┌─────────────────────────────────────────────────────────────────┐
│                    SWOT分析矩阵                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                    内部因素           外部因素                   │
│                                                                 │
│   有利    ┌─────────────────┐     ┌─────────────────┐           │
│           │    S (优势)      │     │    O (机会)      │           │
│           │   Strengths     │     │  Opportunities  │           │
│           │                 │     │                 │           │
│           │ • 技术领先      │     │ • 市场增长      │           │
│           │ • 品牌知名度    │     │ • 政策支持      │           │
│           │ • 团队经验      │     │ • 竞品失误      │           │
│           │ • 资金充裕      │     │ • 技术变革      │           │
│           └─────────────────┘     └─────────────────┘           │
│                                                                 │
│   不利    ┌─────────────────┐     ┌─────────────────┐           │
│           │    W (劣势)      │     │    T (威胁)      │           │
│           │   Weaknesses    │     │    Threats      │           │
│           │                 │     │                 │           │
│           │ • 资源有限      │     │ • 新进入者      │           │
│           │ • 渠道不足      │     │ • 替代品出现    │           │
│           │ • 技术债务      │     │ • 政策收紧      │           │
│           │ • 人才短缺      │     │ • 价格战        │           │
│           └─────────────────┘     └─────────────────┘           │
│                                                                 │
│   策略推导:                                                      │
│   • SO策略: 利用优势抓住机会 (进攻型)                            │
│   • WO策略: 克服劣势把握机会 (扭转型)                            │
│   • ST策略: 利用优势规避威胁 (多元化)                            │
│   • WT策略: 减少劣势回避威胁 (防御型)                            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 波特五力模型

**波特五力模型**分析行业竞争强度和盈利能力。

```
┌─────────────────────────────────────────────────────────────────┐
│                    波特五力模型                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│                    ┌─────────────┐                              │
│                    │  新进入者   │                              │
│                    │  威胁      │                              │
│                    │ (Threat of  │                              │
│                    │  New Entry) │                              │
│                    └──────┬──────┘                              │
│                           │                                     │
│   ┌─────────────┐  ┌──────┴──────┐  ┌─────────────┐            │
│   │  供应商    │  │             │  │   购买者    │            │
│   │  议价能力  │◀─│   行业竞争   │─▶│  议价能力   │            │
│   │ (Supplier  │  │  (Rivalry)  │  │  (Buyer     │            │
│   │  Power)    │  │             │  │   Power)    │            │
│   └─────────────┘  └──────┬──────┘  └─────────────┘            │
│                           │                                     │
│                    ┌──────┴──────┐                              │
│                    │  替代品    │                              │
│                    │  威胁      │                              │
│                    │ (Threat of  │                              │
│                    │  Substitutes)│                             │
│                    └─────────────┘                              │
│                                                                 │
│   分析要点:                                                      │
│   • 五力越强，行业盈利能力越弱                                   │
│   • 找到竞争压力最小的切入点                                     │
│   • 通过差异化降低直接竞争                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

#### 波特五力示例：外卖平台

```
┌─────────────────────────────────────────────────────────────────┐
│                外卖平台行业五力分析                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   维度              强度    分析                                 │
│   ─────────────────────────────────────────────────────────     │
│   行业竞争          极高   美团vs饿了么，烧钱大战，同质化严重    │

│   供应商议价能力    中     餐厅分散，但头部连锁有话语权           │

│   替代品威胁        中     到店消费、预制菜、自炊                 │
│                                                                 │
│   结论: 行业竞争激烈，盈利能力受挤压                             │
│   策略: 差异化(品质外卖)、垂直细分(高端/健康)、生态扩展          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 面试要点

### Q1: 如何做竞品分析？

**答案**：1) 选择直接竞品(解决相同问题)和间接竞品(解决不同问题但目标用户重叠)；2) 从功能、体验、商业模式、数据表现多维度分析；3) 使用功能矩阵对比；4) 总结优劣势和差异化机会。关键是找到"我们可以做得更好"的切入点。

### Q2: 用户画像(Persona)有什么用？

**答案**：用户画像让抽象的"用户"变得具体，帮助团队建立共同语言。在决策时问"小李会喜欢吗"比"用户会喜欢吗"更有效。一个好的画像要包含人口统计、行为特征、目标动机、痛点和常用产品。注意定期更新，避免基于过时假设。

### Q3: SWOT分析中，机会和威胁怎么区分？

**答案**：机会和威胁都是外部因素，机会是有利的外部环境变化，威胁是不利的外部变化。判断标准：这是否在我们的控制范围内？我们能改变的是优势/劣势，不能改变只能应对的是机会/威胁。

### Q4: 什么时候用定性研究，什么时候用定量研究？

**答案**：探索期用定性(访谈、观察)发现问题和洞察；验证期用定量(问卷、数据分析)验证假设的规模；优化期两者结合。简单说：不知道"为什么"时用定性，不知道"有多少"时用定量。
