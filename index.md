## 🎯 学习路线

### 阶段1：工程基础
1. 数据结构：数组 → 链表 → 栈/队列 → 树 → 图
2. 算法：排序 → 二分查找 → 递归 → 动态规划
3. 系统：操作系统基础 → 网络基础

### 阶段2：工程实践
1. 软件工程：设计模式 → 架构 → 测试
2. 数据库：SQL → 索引 → 事务
3. 分布式：CAP → 一致性 → 微服务

### 阶段3：AI系统
1. 机器学习基础
2. 深度学习：CNN → RNN → Transformer
3. 大语言模型与RAG

### 阶段4：进阶主题
1. 编译原理
2. 计算机体系结构
3. 网络安全
