# 优先队列 (Priority Queue)

## 简介

**优先队列（Priority Queue）**是一种抽象数据类型，类似于普通队列，但每个元素都有关联的优先级。出队时，优先级最高的元素先出队，而不是先进先出。优先队列通常使用堆（Heap）实现，以支持高效的操作。

## 基本操作

- **插入（Insert/Push）**：将元素加入队列，时间复杂度 O(log n)
- **提取最大/最小（Extract Max/Min）**：移除并返回优先级最高/最低的元素，时间复杂度 O(log n)
- **查看最大/最小（Peek/Top）**：返回优先级最高/最低的元素但不移除，时间复杂度 O(1)

## 应用场景

- **任务调度**：操作系统优先级调度
- **图算法**：Dijkstra最短路径、Prim最小生成树
- **Top K问题**：查找最大/最小的K个元素
- **数据流处理**：中位数维护、滑动窗口极值
- **外部排序**：K路归并的多路选择

## 实现方式

| 实现方式 | 插入 | 提取极值 | 空间 |
|---------|------|---------|------|
| 有序数组 | O(n) | O(1) | O(n) |
| 有序链表 | O(n) | O(1) | O(n) |
| **二叉堆** | **O(log n)** | **O(log n)** | **O(n)** |
| 二叉搜索树 | O(log n) | O(log n) | O(n) |
