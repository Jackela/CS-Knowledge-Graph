# 链表 (Linked List)

链表是一种线性数据结构，元素通过指针链接而非连续内存存储。与数组不同，链表可以动态增长，不需要预先分配固定大小的内存空间。

## 核心概念

链表由节点组成，每个节点包含：
- **数据域**：存储元素值
- **指针域**：存储下一个节点的地址

### 常见类型

1. **单链表**：每个节点指向下一个节点
2. **双向链表**：每个节点同时指向前驱和后继
3. **循环链表**：尾节点指向头节点，形成环

## 基本操作

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 头插法 | O(1) | 在头部插入新节点 |
| 尾插法 | O(n) | 需要遍历到尾部 |
| 查找 | O(n) | 顺序遍历查找 |
| 删除 | O(n) | 先查找再删除 |

## 与数组的对比

| 特性 | 链表 | 数组 |
|------|------|------|
| 内存分配 | 动态分散 | 连续固定 |
| 随机访问 | O(n) | O(1) |
| 插入删除 | O(1)* | O(n) |
| 缓存友好 | 否 | 是 |

*注：链表插入删除需要先定位到位置，查找过程为O(n)

## 应用场景

- 实现栈和队列
- 内存管理（空闲内存块链表）
- 文件系统的簇链
- 图的邻接表表示
