# 位运算 (Bitwise Operations)

位运算是对二进制数的各位进行操作的运算，是计算机底层操作的基础。

## 基本操作

| 运算 | 符号 | 说明 | 示例 (4位) |
|------|------|------|-----------|
| 与 (AND) | `&` | 两位都为1时结果为1 | `1010 & 1100 = 1000` |
| 或 (OR) | `\|` | 任一位为1时结果为1 | `1010 \| 1100 = 1110` |
| 异或 (XOR) | `^` | 两位不同时结果为1 | `1010 ^ 1100 = 0110` |
| 非 (NOT) | `~` | 位取反 | `~1010 = 0101` |
| 左移 | `<<` | 各位向左移动，右侧补0 | `1010 << 1 = 0100` |
| 右移 | `>>` | 各位向右移动，左侧补符号位 | `1010 >> 1 = 0101` |
| 无符号右移 | `>>>` | 各位向右移动，左侧补0 | `1010 >>> 1 = 0101` |

## 常见应用

### 1. 判断奇偶
```
if (n & 1) == 0:  // 偶数
if (n & 1) == 1:  // 奇数
```

### 2. 交换两个数
```
a = a ^ b
b = a ^ b  
a = a ^ b
```

### 3. 清零最低位的1
```
n = n & (n - 1)
// 用于统计1的个数、判断是否为2的幂
```

### 4. 获取最低位的1
```
lowbit = n & (-n)
// 树状数组核心操作
```
