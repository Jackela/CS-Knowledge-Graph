# 完全二叉树 (Complete Binary Tree)

## 简介

**完全二叉树（Complete Binary Tree）**是一种特殊的二叉树，除了最后一层外，每一层都被完全填满，并且最后一层的节点都集中在左侧。这种结构使得完全二叉树可以用数组高效存储，是堆数据结构的基础。

## 性质

- 除最后一层外，每层节点数达到最大值
- 最后一层节点从左到右连续填充
- 具有 $n$ 个节点的完全二叉树深度为 $\lfloor \log_2 n \rfloor + 1$
- 适合用数组存储，父子关系通过索引计算

## 数组存储

对于索引从 0 开始的数组：
- 节点 i 的左子节点：2i + 1
- 节点 i 的右子节点：2i + 2
- 节点 i 的父节点：⌊(i-1)/2⌋
