# 约定

软件系统的很多行为并不是被硬性规则强制出来的，而是人或框架共同假定的一组默认规则。

约定是一个在系统边界内被普遍接受的默认假设，用来简化显式配置。

性质：

* 非强制（可以覆盖）；

* 可共享（依赖共同理解）；

* 用来降低描述和协调成本。
