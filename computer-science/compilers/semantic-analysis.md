# 语义分析 (Semantic Analysis)

## 简介

**语义分析**是编译器的第三个阶段，负责检查程序的语义正确性。它在语法分析生成的 AST 基础上，进行类型检查、作用域分析、控制流检查等，确保程序不仅语法正确，而且在语义上有意义。语义分析还负责符号表管理和中间代码生成。

## 核心概念

| 概念 | 英文 | 说明 |
|------|------|------|
| **Symbol Table** | 符号表 | 存储变量、函数等标识符信息的数据结构 |
| **Type Checking** | 类型检查 | 验证操作数类型是否兼容 |
| **Scope Analysis** | 作用域分析 | 确定标识符的可见范围和生命周期 |
| **Attribute Grammar** | 属性文法 | 为文法符号附加属性的形式化方法 |
| **S-Attributed** | S-属性 | 仅使用综合属性的属性文法 |
| **L-Attributed** | L-属性 | 允许继承属性的属性文法 |
| **Control Flow** | 控制流分析 | 检查程序控制结构的正确性 |

# symbol_table.py
from dataclasses import dataclass, field
from typing import Dict, Optional, List, Any
from enum import Enum, auto

class SymbolKind(Enum):
    VARIABLE = auto()
    FUNCTION = auto()
    PARAMETER = auto()
    CLASS = auto()
    TYPE = auto()

@dataclass
class Symbol:
    name: str
    kind: SymbolKind
    type: str
    scope_level: int
    line: int
    attributes: Dict[str, Any] = field(default_factory=dict)

class SymbolTable:
    \"\"\"符号表，支持嵌套作用域\"\"\"\n    \n    def __init__(self):\n        self.scopes: List[Dict[str, Symbol]] = [{}]  # 全局作用域\n        self.scope_level = 0\n    \n    def enter_scope(self):\n        \"\"\"进入新作用域\"\"\"\n        self.scopes.append({})\n        self.scope_level += 1\n    \n    def exit_scope(self):\n        \"\"\"退出当前作用域\"\"\"\n        if self.scope_level > 0:\n            self.scopes.pop()\n            self.scope_level -= 1\n    \n    def define(self, name: str, kind: SymbolKind, type_: str, \n               line: int, **attrs) -> Symbol:\n        \"\"\"定义符号\"\"\"\n        if name in self.scopes[-1]:\n            raise NameError(f'{name} already defined in current scope')\n        \n        symbol = Symbol(name, kind, type_, self.scope_level, line, attrs)\n        self.scopes[-1][name] = symbol\n        return symbol\n    \n    def lookup(self, name: str) -> Optional[Symbol]:\n        \"\"\"查找符号（从内到外）\"\"\"\n        for scope in reversed(self.scopes):\n            if name in scope:\n                return scope[name]\n        return None\n    \n    def lookup_current(self, name: str) -> Optional[Symbol]:\n        \"\"\"只在当前作用域查找\"\"\"\n        return self.scopes[-1].get(name)\n    \n    def display(self):\n        \"\"\"显示符号表内容\"\"\"\n        print(\"Symbol Table:\")\n        for level, scope in enumerate(self.scopes):\n            print(f'  Level {level}:')\n            for name, sym in scope.items():\n                print(f'    {name}: {sym.kind.name} ({sym.type})')\n\n# 使用示例\nif __name__ == '__main__':\n    st = SymbolTable()\n    \n    # 全局变量\n    st.define('x', SymbolKind.VARIABLE, 'int', 1)\n    st.define('main', SymbolKind.FUNCTION, 'void', 2)\n    \n    # 函数作用域\n    st.enter_scope()\n    st.define('y', SymbolKind.VARIABLE, 'int', 3)\n    st.define('z', SymbolKind.PARAMETER, 'float', 3)\n    \n    # 嵌套作用域\n    st.enter_scope()\n    st.define('y', SymbolKind.VARIABLE, 'string', 5)  # 遮蔽外层 y\n    \n    print(f\"Lookup 'y': {st.lookup('y')}\")  # 找到内层 y\n    print(f\"Lookup 'x': {st.lookup('x')}\")  # 找到全局 x\n    \n    st.display()\n```\n\n### 2. 类型检查器\n\n```python\n# type_checker.py\nfrom dataclasses import dataclass\nfrom typing import Dict, Set, Optional\nfrom enum import Enum, auto\n
