# 动态规划 (Dynamic Programming)

动态规划（DP）是一种通过将复杂问题分解为重叠子问题来求解的算法思想。它通过存储子问题的解（记忆化）避免重复计算，将指数级复杂度优化为多项式级。

## 核心思想

### 最优子结构
问题的最优解包含子问题的最优解。
- 整体最优 = 局部最优的组合

### 重叠子问题
子问题被重复计算多次。
- 存储子问题解避免重复计算

### 无后效性
当前状态只与过去有关，与将来无关。
- 状态定义必须满足此性质

## DP解题步骤

1. **定义状态**：确定dp数组含义
2. **状态转移方程**：找出状态间的关系
3. **初始化**：确定边界条件
4. **计算顺序**：确定遍历方向
5. **空间优化**：滚动数组等技巧

## 经典问题

### 线性DP
| 问题 | 状态定义 | 转移方程 |
|------|---------|---------|
| 爬楼梯 | dp[i] = 爬i阶的方法数 | dp[i] = dp[i-1] + dp[i-2] |
| 最长递增子序列 | dp[i] = 以i结尾的LIS长度 | dp[i] = max(dp[j]+1), j<i |
| 最大子数组和 | dp[i] = 以i结尾的最大和 | dp[i] = max(nums[i], dp[i-1]+nums[i]) |

### 背包问题
| 类型 | 特点 | 转移方程 |
|------|------|---------|
| 0/1背包 | 物品只能用一次 | dp[i][w] = max(dp[i-1][w], dp[i-1][w-weight[i]]+value[i]) |
| 完全背包 | 物品无限用 | dp[w] = max(dp[w], dp[w-weight[i]]+value[i]) |
| 多重背包 | 物品有限个 | 二进制优化或单调队列 |

### 区间DP
- 石子合并
- 矩阵链乘法
- 最长回文子序列

### 状态压缩DP
- 旅行商问题（TSP）
- 棋盘覆盖问题

## 空间优化

### 滚动数组
```
// 一维优化0/1背包
for i in items:
    for w from W down to weight[i]:
        dp[w] = max(dp[w], dp[w-weight[i]]+value[i])
```

### 状态压缩
使用位运算表示集合状态，降低维度。

## 与贪心算法的区别

| 特性 | 动态规划 | 贪心算法 |
|------|---------|---------|
| 选择策略 | 考虑所有子问题 | 局部最优选择 |
| 回退 | 可以回退选择 | 不回退 |
| 适用性 | 更广泛 | 需要贪心选择性质 |
| 典型问题 | 背包、LIS | 活动选择、哈夫曼编码 |
