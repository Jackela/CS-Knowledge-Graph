# 比特

计算机硬件只会区分两种物理[状态](./state.md)——导通 / 断开、电压高 / 低、磁极正 / 负。
要让这些状态可被逻辑系统识别，就必须赋予它们一个抽象的名字：

“这两种状态” → 0 与 1（[二元](./binary.md)）

于是得到计算机[信息](./data.md)的最小可区分单位：bit（[比特](./bit.md)）。

bit 是“二元信息单元”

| 概念           | 含义                      |
| ------------ | ----------------------- |
| **[比特](./bit.md)**  | 1 或 0 的单个状态             |
| **[字节](./byte.md)** | 通常为 8 bit 的最小可[寻址](./memory-addressing.md)单元      |
| **[字](./word.md)**  | 由若干**[字节](./byte.md)组成，是 [CPU](./cpu.md) 一次操作的基本宽度 |
