# 数据 (Data)

数据是信息的载体，是可以被计算机识别、存储和处理的符号集合。

## 数据的分类

### 按数据类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 结构化数据 | 具有固定格式，存储在关系数据库中 | 数据库表、Excel |
| 半结构化数据 | 有一定结构但不严格 | JSON、XML、日志 |
| 非结构化数据 | 无预定义格式 | 文本、图片、视频 |

### 按数据规模
| 规模 | 量级 | 处理方式 |
|------|------|---------|
| 小规模 | KB-MB | 单机内存处理 |
| 中规模 | GB-TB | 数据库/大数据框架 |
| 大规模 | PB-EB | 分布式存储与计算 |

## 数据在计算机中的表示

### 二进制表示
```
所有数据最终以二进制存储：
- 整数：补码表示
- 浮点数：IEEE 754标准
- 字符：ASCII/Unicode编码
- 多媒体：特定编码格式
```

### 存储层次
```
寄存器 → 缓存(L1/L2/L3) → 内存 → 磁盘 → 分布式存储
  ↑                                              ↑
速度快、容量小                              速度慢、容量大
```

## 数据与信息

| 概念 | 定义 | 关系 |
|------|------|------|
| 数据 | 原始符号记录 | 信息的载体 |
| 信息 | 加工后的数据 | 数据的语义解释 |
| 知识 | 信息的归纳总结 | 从信息中提炼 |
