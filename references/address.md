# 地址

计算机的物理世界只有存储单元。
要在程序中"找到"某个单元，就需要给它一个可被引用的名字。

地址是机器可识别的"位置标识符"

一种从地址集合到存储单元集合的映射。
每个地址唯一对应一个可读写的[存储单元](./storage-cell.md)。

| 层次                         | 地址含义         | 举例          |
| -------------------------- | ------------ | ----------- |
| **物理地址** | 硬件总线上的真实位置   | DRAM 中的单元编号 |
| **线性地址**   | 连续可加的逻辑位置    | [数组](../computer-science/data-structures/array.md) |
| **虚拟地址**  | 由操作系统映射的抽象位置 | [进程](../computer-science/systems/process.md)视角下的指针    |
| **符号地址** | 编译器/链接器分配的名字 | 变量名、函数名     |
