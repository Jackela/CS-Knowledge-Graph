# 离散数学 (Discrete Mathematics)

## 简介

**离散数学 (Discrete Mathematics)** 是研究离散对象（非连续）的数学分支，与连续数学（如微积分）相对。它是计算机科学的数学基础，为算法设计、数据结构、密码学、编译原理、数据库理论等领域提供形式化工具和思维方式。

离散数学的核心特点是研究**可数集合**上的结构和关系，包括整数、图、逻辑命题、有限状态等。与连续数学关注极限、导数和积分不同，离散数学关注组合、枚举、关系和网络。

## 核心领域

### 集合论 (Set Theory)

集合论是离散数学的基础，由康托尔 (Georg Cantor) 在19世纪末创立。现代数学几乎所有分支都建立在集合论之上。

#### 基本概念

```
集合 (Set)：对象的汇集，对象称为元素

表示方法：
- 列举法：A = {1, 2, 3, 4, 5}
- 描述法：B = {x | x 是偶数且 x < 10}
- 文氏图 (Venn Diagram)

常见集合符号：
ℕ：自然数集 {0, 1, 2, 3, ...}
ℤ：整数集 {..., -2, -1, 0, 1, 2, ...}
ℚ：有理数集
ℝ：实数集
∅：空集

元素与集合的关系：
- a ∈ A：a 是 A 的元素
- a ∉ A：a 不是 A 的元素
```

#### 集合运算

```
┌─────────────────────────────────────────────────────────────┐
│                      集合运算                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   并集 (Union)：A ∪ B = {x | x ∈ A 或 x ∈ B}               │
│                                                             │
│       A       B              A ∪ B                          │
│     ┌───┐   ┌───┐          ┌───────┐                       │
│     │███│███│   │    →     │███████│                       │
│     └───┘   └───┘          └───────┘                       │
│                                                             │
│   交集 (Intersection)：A ∩ B = {x | x ∈ A 且 x ∈ B}       │
│                                                             │
│       A       B              A ∩ B                          │
│     ┌───┐   ┌───┐          ┌───┐                           │
│     │███│███│   │    →     │███│                           │
│     └───┘   └───┘          └───┘                           │
│                                                             │
│   差集 (Difference)：A - B = {x | x ∈ A 且 x ∉ B}         │
│                                                             │
│       A       B              A - B                          │
│     ┌───┐   ┌───┐          ┌───┐                           │
│     │███│███│   │    →     │███│                           │
│     └───┘   └───┘          └───┘                           │
│                                                             │
│   补集 (Complement)：Ā = {x | x ∈ U 且 x ∉ A}             │
│                                                             │
└─────────────────────────────────────────────────────────────┘

集合运算性质：
- 交换律：A ∪ B = B ∪ A，A ∩ B = B ∩ A
- 结合律：(A ∪ B) ∪ C = A ∪ (B ∪ C)
- 分配律：A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
- 德摩根律：A ∩ B = Ā ∪ B̄，A ∪ B = Ā ∩ B̄
```

#### 幂集与笛卡尔积

```
幂集 (Power Set)：
P(A) = {X | X ⊆ A}，即 A 的所有子集组成的集合

若 |A| = n，则 |P(A)| = 2ⁿ

示例：A = {1, 2}
P(A) = {∅, {1}, {2}, {1, 2}}

笛卡尔积 (Cartesian Product)：
A × B = {(a, b) | a ∈ A, b ∈ B}

示例：A = {1, 2}, B = {a, b}
A × B = {(1,a), (1,b), (2,a), (2,b)}
|A × B| = |A| × |B|
```

### 逻辑学 (Logic)

逻辑学研究推理的形式结构和有效论证。在计算机科学中，逻辑是电路设计、程序验证、数据库查询和人工智能的基础。

#### 命题逻辑

```
命题 (Proposition)：能判断真假的陈述句

- 真值：真 (T/1) 或 假 (F/0)
- 原子命题：不可再分解的基本命题
- 复合命题：由逻辑联结词连接而成

逻辑联结词：
┌─────────┬─────────┬─────────────────────────────┐
│  符号   │  名称   │         含义                 │
├─────────┼─────────┼─────────────────────────────┤
│    ¬    │  非     │  ¬P：P 的否定               │
│    ∧    │  合取   │  P ∧ Q：P 且 Q              │
│    ∨    │  析取   │  P ∨ Q：P 或 Q              │
│    →    │  蕴含   │  P → Q：若 P 则 Q           │
│    ↔    │  等价   │  P ↔ Q：P 当且仅当 Q        │
└─────────┴─────────┴─────────────────────────────┘

真值表：
┌───┬───┬───┬───┬───┬───┐
│ P │ Q │¬P │P∧Q│P∨Q│P→Q│
├───┼───┼───┼───┼───┼───┤
│ T │ T │ F │ T │ T │ T │
│ T │ F │ F │ F │ T │ F │
│ F │ T │ T │ F │ T │ T │
│ F │ F │ T │ F │ F │ T │
└───┴───┴───┴───┴───┴───┘

重要等价式：
- P → Q ≡ ¬P ∨ Q
- P ↔ Q ≡ (P → Q) ∧ (Q → P)
```

#### 谓词逻辑

```
谓词 (Predicate)：含变量的命题 P(x)

示例：
P(x) = "x 是偶数"
P(2) = 真，P(3) = 假

量词：
- 全称量词 ∀：∀x P(x) 表示"对所有 x，P(x) 成立"
- 存在量词 ∃：∃x P(x) 表示"存在 x，使 P(x) 成立"

量词否定：
- ¬∀x P(x) ≡ ∃x ¬P(x)
- ¬∃x P(x) ≡ ∀x ¬P(x)

示例翻译：
"所有人都会死，苏格拉底是人，所以苏格拉底会死"

设：M(x)="x 是人"，D(x)="x 会死"，s="苏格拉底"

前提：∀x (M(x) → D(x))，M(s)
结论：D(s)
```

### 关系与函数

#### 二元关系

```
二元关系 (Binary Relation)：
R ⊆ A × B，A 到 B 的关系是笛卡尔积的子集

关系的表示：
1. 集合表示：R = {(1,a), (2,b), (3,a)}
2. 矩阵表示：布尔矩阵 M_R
3. 有向图表示

关系性质（设 R 是 A 上的关系）：
┌──────────┬────────────────────────────────────┐
│  性质    │            定义                     │
├──────────┼────────────────────────────────────┤
│ 自反性   │ ∀a∈A: (a,a) ∈ R                    │
│ 反自反性 │ ∀a∈A: (a,a) ∉ R                    │
│ 对称性   │ (a,b)∈R → (b,a)∈R                  │
│ 反对称性 │ (a,b)∈R ∧ (b,a)∈R → a=b            │
│ 传递性   │ (a,b)∈R ∧ (b,c)∈R → (a,c)∈R        │
└──────────┴────────────────────────────────────┘

等价关系：自反、对称、传递
偏序关系：自反、反对称、传递
全序关系：偏序 + 任意两元素可比
```

#### 函数

```
函数 (Function)：特殊的二元关系 f: A → B

性质：
- 每个 A 中元素恰对应一个 B 中元素
- 记作：f(a) = b 或 (a,b) ∈ f

函数类型：
- 单射 (Injective)：f(a₁) = f(a₂) → a₁ = a₂
- 满射 (Surjective)：∀b∈B, ∃a∈A: f(a) = b
- 双射 (Bijective)：既是单射又是满射

函数的复合：
(f ∘ g)(x) = f(g(x))

反函数：
若 f 是双射，则存在反函数 f⁻¹
```

### 组合数学 (Combinatorics)

组合数学研究离散结构的计数、枚举和优化问题。详见 [组合数学](./combinatorics.md)。

```

2. 排列与组合
```

### 代数结构 (Algebraic Structures)

代数结构是带有运算的集合，研究运算的性质和结构。

#### 群论基础

```
群 (Group)：(G, *)，满足四个性质：

1. 封闭性：∀a,b∈G: a * b ∈ G
2. 结合律：(a * b) * c = a * (b * c)
3. 单位元：∃e∈G: a * e = e * a = a
4. 逆元：∀a∈G, ∃a⁻¹∈G: a * a⁻¹ = e

示例：
- (ℤ, +)：整数加法群
- (ℝ\{0}, ×)：非零实数乘法群

特殊群：
- 阿贝尔群：满足交换律 a * b = b * a
- 循环群：由单个元素生成
- 置换群：由置换（排列）组成

子群：H ⊆ G 且 (H, *) 也是群

拉格朗日定理：
若 H 是 G 的子群，则 |H| 整除 |G|
```

#### 环与域

```
环 (Ring)：(R, +, ·)，满足：
- (R, +) 是阿贝尔群
- (R, ·) 是半群（封闭、结合）
- 乘法对加法分配律

示例：
- (ℤ, +, ×)：整数环
- 多项式环

域 (Field)：(F, +, ·)，满足：
- (F, +) 是阿贝尔群
- (F\{0}, ·) 是阿贝尔群
- 分配律

示例：
- ℚ：有理数域
- ℝ：实数域
- ℤ/pℤ：p 元有限域（p 为素数）

有限域在密码学中有重要应用。
```

### 布尔代数 (Boolean Algebra)

布尔代数是数字电路和计算机逻辑的基础。详见 [布尔代数](../references/bit.md)。

```
布尔代数：(B, ∧, ∨, ¬, 0, 1)

公理：
1. 交换律：a ∧ b = b ∧ a，a ∨ b = b ∨ a
2. 结合律：(a ∧ b) ∧ c = a ∧ (b ∧ c)
3. 分配律：a ∧ (b ∨ c) = (a ∧ b) ∨ (a ∧ c)
4. 同一律：a ∧ 1 = a，a ∨ 0 = a
5. 补律：a ∧ ¬a = 0，a ∨ ¬a = 1

德摩根律：
- ¬(a ∧ b) = ¬a ∨ ¬b
- ¬(a ∨ b) = ¬a ∧ ¬b

应用：
- 数字电路设计
- 逻辑门实现
- 布尔表达式简化
```

### 数论基础 (Number Theory)

数论研究整数的性质，在密码学中有核心应用。

```
整除与模运算：

a | b：a 整除 b（b = ka）

带余除法：
a = qb + r，其中 0 ≤ r < b

最大公约数：gcd(a, b)
最小公倍数：lcm(a, b)

欧几里得算法：
gcd(a, b) = gcd(b, a mod b)

扩展欧几里得算法：
求 x, y 使得 ax + by = gcd(a, b)

模运算：
a ≡ b (mod m) 表示 m | (a - b)

模运算性质：
- (a + b) mod m = ((a mod m) + (b mod m)) mod m
- (a × b) mod m = ((a mod m) × (b mod m)) mod m

费马小定理：
若 p 是素数，p ∤ a，则 a^(p-1) ≡ 1 (mod p)

欧拉定理：
若 gcd(a, n) = 1，则 a^φ(n) ≡ 1 (mod n)

RSA加密算法的基础。
```

## 面试要点

### 常见问题

**Q1: 什么是等价关系？给出一个例子。**

> 等价关系是满足自反性、对称性和传递性的二元关系。
>
> 例子：整数集上的"模 n 同余"关系
> - 自反：a ≡ a (mod n)
> - 对称：a ≡ b → b ≡ a
> - 传递：a ≡ b 且 b ≡ c → a ≡ c

**Q2: 解释鸽巢原理及其应用。**

> 鸽巢原理：将 n+1 个物体放入 n 个盒子，至少有一个盒子包含不少于2个物体。
>
> 应用：
> - 证明：任意 n+1 个整数中必有两个数之差能被 n 整除
> - 哈希冲突的必然性
> - 生日问题：23人中两人生日相同的概率 > 50%

**Q3: 如何判断一个图是否为二分图？**

> 二分图判定：图是二分图当且仅当图中不存在奇数长度的环。
>
> 算法：BFS/DFS 双色染色
> 1. 任选起点，染为颜色A
> 2. 将所有邻居染为颜色B
> 3. 继续染色，若发现相邻顶点同色，则不是二分图
> 4. 若染色完成无冲突，则是二分图

**Q4: 命题逻辑与谓词逻辑的区别？**

> 命题逻辑：
> - 原子命题是不可再分的基本单位
> - 无法表达"所有"、"存在"等量词
>
> 谓词逻辑：
> - 引入变量和谓词，可表达更复杂的陈述
> - 支持全称量词 ∀ 和存在量词 ∃
> - 表达能力更强，是数学逻辑的基础

**Q5: 解释德摩根律及其在编程中的应用。**

> 德摩根律：
> - ¬(A ∧ B) ≡ ¬A ∨ ¬B
> - ¬(A ∨ B) ≡ ¬A ∧ ¬B
>
> 编程应用：
> - 逻辑表达式简化
> - 条件判断优化
> - 布尔查询转换（如 SQL WHERE 子句）

**Q6: 什么是偏序关系？举例说明。**

> 偏序关系是满足自反性、反对称性和传递性的关系。
>
> 例子：
> - 集合的包含关系 ⊆
> - 整数的整除关系 |
> - 任务的依赖关系（前置任务）
>
> 偏序不要求任意两元素可比，全序是其特例。

**Q7: 证明：n 个顶点的树有 n-1 条边。**

> 数学归纳法证明：
> - 基础：n=1 时，边数为0，成立
> - 归纳：假设对 n=k 成立，考虑 n=k+1
> - 树中必有叶子节点（度为1）
> - 删除叶子节点及其边，得到 k 个顶点的树
> - 由归纳假设，剩余树有 k-1 条边
> - 原树有 (k-1) + 1 = k 条边，证毕

**Q8: 解释 RSA 加密的基本数学原理。**

> RSA 基于数论中的模运算和欧拉定理：
> 1. 选择两个大素数 p, q，计算 n = pq
> 2. 计算 φ(n) = (p-1)(q-1)
> 3. 选择 e 使得 gcd(e, φ(n)) = 1
> 4. 计算 d 使得 ed ≡ 1 (mod φ(n))
> 5. 公钥：(e, n)，私钥：(d, n)
> 6. 加密：c = m^e mod n
> 7. 解密：m = c^d mod n
>
> 安全性基于大整数分解的困难性。

**Q9: 容斥原理如何计算三个集合的并集大小？**

> |A ∪ B ∪ C| = |A| + |B| + |C|
>             - |A∩B| - |B∩C| - |A∩C|
>             + |A∩B∩C|
>
> 应用：计算"至少满足一个条件"的元素数量
> 例如：1-100中能被2或3或5整除的数的个数

**Q10: 什么是函数的复合？什么情况下函数有反函数？**

> 函数复合：(f ∘ g)(x) = f(g(x))
> - 复合满足结合律：(f ∘ g) ∘ h = f ∘ (g ∘ h)
> - 一般不满足交换律
>
> 反函数存在条件：函数必须是双射（单射且满射）
> - 单射：不同的输入产生不同的输出
> - 满射：值域等于到达域
> - 双射保证每个输出对应唯一的输入
