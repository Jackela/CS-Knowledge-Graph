## 基础概念

### 样本空间与事件

```
样本空间 (Sample Space)：
随机试验所有可能结果的集合，记为 Ω

示例：掷骰子
Ω = {1, 2, 3, 4, 5, 6}

事件 (Event)：
样本空间的子集

- 基本事件：{1}, {2}, ..., {6}
- 复合事件：偶数点 = {2, 4, 6}
- 必然事件：Ω
- 不可能事件：∅

事件运算：
- 并 A∪B：A或B发生
- 交 A∩B：A和B都发生
- 补 Aᶜ：A不发生
- 差 A-B：A发生但B不发生
```

### 概率的定义

```
古典概型（等可能）：
P(A) = |A| / |Ω| = 有利结果数 / 总结果数

示例：掷骰子得到偶数的概率
P(偶数) = 3/6 = 1/2

频率学派：
P(A) = lim(n→∞) n(A)/n
事件发生的频率极限

公理化定义（Kolmogorov）：
1. 非负性：P(A) ≥ 0
2. 规范性：P(Ω) = 1
3. 可列可加性：互斥事件，P(∪Aᵢ) = ΣP(Aᵢ)
```

## 条件概率

### 条件概率公式

```
条件概率：
P(A|B) = P(A∩B) / P(B)  (P(B) > 0)

直观：在B发生的条件下，A发生的概率

乘法公式：
P(A∩B) = P(A) × P(B|A) = P(B) × P(A|B)

链式法则：
P(A₁∩A₂∩...∩Aₙ) = P(A₁) × P(A₂|A₁) × P(A₃|A₁∩A₂) × ...

示例：
袋中有3红2蓝共5个球，不放回抽2个
P(第1红) = 3/5
P(第2红|第1红) = 2/4 = 1/2
P(两红) = 3/5 × 2/4 = 3/10
```

### 全概率公式

```
全概率公式：
若 B₁, B₂, ..., Bₙ 构成样本空间的划分（互斥且穷尽）

P(A) = Σ P(A|Bᵢ) × P(Bᵢ)

应用场景：
当直接计算P(A)困难，但知道在不同条件下A的概率时

示例：
工厂有两台机器，甲生产60%产品，乙生产40%
甲的次品率2%，乙的次品率3%

P(次品) = P(次品|甲)P(甲) + P(次品|乙)P(乙)
        = 0.02 × 0.6 + 0.03 × 0.4
        = 0.012 + 0.012 = 0.024
```

### 贝叶斯定理

```
贝叶斯定理：
P(A|B) = P(B|A) × P(A) / P(B)

        = P(B|A) × P(A) / [P(B|A)P(A) + P(B|Aᶜ)P(Aᶜ)]

术语：
- P(A)：先验概率（Prior）
- P(A|B)：后验概率（Posterior）
- P(B|A)：似然（Likelihood）
- P(B)：证据（Evidence）

示例：医学检测
某病患病率1%，检测准确率：
- 患者阳性率99%（真阳性）
- 健康者阴性率95%（真阴性）

问：检测阳性时，实际患病的概率？

P(病|阳) = P(阳|病)P(病) / [P(阳|病)P(病) + P(阳|健康)P(健康)]
         = 0.99 × 0.01 / (0.99 × 0.01 + 0.05 × 0.99)
         = 0.0099 / 0.0594 ≈ 16.7%

反直觉：即使检测准确率高，由于患病率低，阳性者实际患病率只有16.7%
```

## 随机变量

### 离散型随机变量

```
概率质量函数 (PMF)：
P(X = x) = p(x)

性质：
1. p(x) ≥ 0
2. Σ p(x) = 1

常见离散分布：

1. 伯努利分布 Bernoulli(p)
   P(X=1) = p, P(X=0) = 1-p
   
2. 二项分布 Binomial(n, p)
   P(X=k) = C(n,k) × pᵏ × (1-p)ⁿ⁻ᵏ
   n次独立伯努利试验成功k次的概率
   
3. 几何分布 Geometric(p)
   P(X=k) = (1-p)ᵏ⁻¹ × p
   首次成功所需的试验次数
   
4. 泊松分布 Poisson(λ)
   P(X=k) = λᵏ × e⁻ˡ / k!
   单位时间/空间内随机事件发生次数
```

### 连续型随机变量

```
概率密度函数 (PDF)：
f(x) ≥ 0,  ∫f(x)dx = 1

概率：P(a ≤ X ≤ b) = ∫ₐᵇ f(x)dx

分布函数 (CDF)：
F(x) = P(X ≤ x) = ∫₋∞ˣ f(t)dt

常见连续分布：

1. 均匀分布 Uniform(a, b)
   f(x) = 1/(b-a),  a ≤ x ≤ b
   
2. 指数分布 Exponential(λ)
   f(x) = λe⁻ˡˣ,  x ≥ 0
   描述泊松过程中事件间隔时间
   
3. 正态分布 Normal(μ, σ²)
   f(x) = (1/√(2π)σ) × e^(-(x-μ)²/(2σ²))
   
   标准正态：Z ~ N(0, 1)
   
4. 高斯分布的性质：
   - 68-95-99.7规则
   - 中心极限定理的基础
```

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# 概率密度函数
x = np.linspace(-4, 4, 1000)
y = stats.norm.pdf(x, 0, 1)  # 标准正态
plt.plot(x, y, label='N(0,1)')

# 累积分布函数
y_cdf = stats.norm.cdf(x, 0, 1)
plt.plot(x, y_cdf, label='CDF')

plt.legend()
plt.show()
```

## 期望与方差

### 期望值

```
定义：
离散：E[X] = Σ x × P(X=x)
连续：E[X] = ∫ x × f(x) dx

性质：
1. E[aX + b] = aE[X] + b
2. E[X + Y] = E[X] + E[Y]
3. 若X,Y独立：E[XY] = E[X]E[Y]

常见分布的期望：
- 伯努利(p)：p
- 二项(n,p)：np
- 泊松(λ)：λ
- 均匀(a,b)：(a+b)/2
- 指数(λ)：1/λ
- 正态(μ,σ²)：μ

条件期望：
E[X|Y=y] = Σ x × P(X=x|Y=y)
```

### 方差与标准差

```
定义：
Var(X) = E[(X - E[X])²] = E[X²] - (E[X])²

标准差：σ = √Var(X)

性质：
1. Var(aX + b) = a²Var(X)
2. 若X,Y独立：Var(X + Y) = Var(X) + Var(Y)

协方差：
Cov(X, Y) = E[(X-E[X])(Y-E[Y])] = E[XY] - E[X]E[Y]

相关系数：
ρ(X, Y) = Cov(X, Y) / (σₓσᵧ)  ∈ [-1, 1]

常见分布的方差：
- 伯努利(p)：p(1-p)
- 二项(n,p)：np(1-p)
- 泊松(λ)：λ
- 均匀(a,b)：(b-a)²/12
- 指数(λ)：1/λ²
- 正态(μ,σ²)：σ²
```

## 大数定律与中心极限定理

### 大数定律

```
弱大数定律：
设 X₁, X₂, ..., Xₙ 是i.i.d.随机变量，E[Xᵢ] = μ

则样本均值 X̄ₙ = (X₁ + ... + Xₙ)/n

对于任意 ε > 0：
lim P(|X̄ₙ - μ| > ε) = 0  (n → ∞)

即：样本均值依概率收敛于期望值

直观：试验次数越多，频率越接近概率

应用：蒙特卡洛方法估计π
```

### 中心极限定理

```
中心极限定理 (CLT)：
设 X₁, X₂, ..., Xₙ 是i.i.d.随机变量，
E[Xᵢ] = μ, Var(Xᵢ) = σ²

则：
(X̄ₙ - μ) / (σ/√n)  →  N(0, 1)  (n → ∞)

即：标准化后的样本均值收敛于标准正态分布

意义：
- 解释了为什么正态分布如此普遍
- 为统计推断提供理论基础
- 无论总体分布如何，样本均值都近似正态

应用：
- 置信区间估计
- 假设检验
- A/B测试样本量计算
```

```python
import numpy as np
import matplotlib.pyplot as plt

# 验证中心极限定理
# 用均匀分布验证
n_samples = 10000
sample_size = 30

# 生成样本均值
sample_means = []
for _ in range(n_samples):
    sample = np.random.uniform(0, 1, sample_size)
    sample_means.append(np.mean(sample))

# 理论：均匀分布U(0,1)的期望=0.5，方差=1/12
# 样本均值的标准差 = sqrt((1/12)/30) ≈ 0.053

# 标准化
standardized = (np.array(sample_means) - 0.5) / np.sqrt(1/12/30)

plt.hist(standardized, bins=50, density=True, alpha=0.7, label='Sample Means')
x = np.linspace(-4, 4, 100)
plt.plot(x, (1/np.sqrt(2*np.pi)) * np.exp(-x**2/2), 
         'r-', label='N(0,1)')
plt.legend()
plt.show()
```

## 马尔可夫链

```
马尔可夫链：具有马尔可夫性质的随机过程

马尔可夫性质：
P(Xₙ₊₁ = x | Xₙ = xₙ, ..., X₀ = x₀) = P(Xₙ₊₁ = x | Xₙ = xₙ)

即：未来状态只依赖于当前状态，与过去无关

转移矩阵：
P = [pᵢⱼ], 其中 pᵢⱼ = P(Xₙ₊₁ = j | Xₙ = i)

性质：
1. pᵢⱼ ≥ 0
2. Σⱼ pᵢⱼ = 1

平稳分布：
π = πP
若存在平稳分布，长期行为将收敛于此

应用：
- PageRank算法
- 自然语言处理（隐马尔可夫模型）
- 推荐系统
- 队列理论
```

## 信息论基础

### 熵

```
信息熵 (Entropy)：
H(X) = -Σ P(x) × log₂P(x)

意义：随机变量的不确定性度量
单位：比特（bits）

性质：
1. H(X) ≥ 0
2. 均匀分布时熵最大
3. 确定事件熵为0

示例：
公平硬币：H = -0.5log₀.₅0.5 - 0.5log₀.₅0.5 = 1 bit
有偏硬币(p=0.9)：H ≈ 0.47 bits

联合熵：
H(X, Y) = -Σ P(x,y) log P(x,y)

条件熵：
H(Y|X) = Σ P(x) H(Y|X=x)

互信息：
I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)

意义：知道Y后，X的不确定性的减少量
```

## 面试要点

### 常见问题

**Q1: 条件概率和联合概率的关系？**
> P(A|B) = P(A∩B)/P(B)，即条件概率是联合概率相对于条件事件概率的归一化。乘法公式：P(A∩B) = P(A)P(B|A) = P(B)P(A|B)。

**Q2: 贝叶斯定理的实际应用？**> 医学诊断（计算患病概率）、垃圾邮件过滤（朴素贝叶斯）、机器学习（贝叶斯推断）、A/B测试结果分析等。

**Q3: 大数定律和中心极限定理的区别？**> 大数定律说明样本均值收敛于期望值；中心极限定理说明样本均值的分布收敛于正态分布。前者讲收敛到什么值，后者讲收敛到什么分布。

**Q4: 为什么正态分布如此重要？**> 1) 中心极限定理保证了许多独立随机变量和的分布趋近正态；2) 数学性质优良，便于分析；3) 自然界许多现象近似正态；4) 统计推断的基础。

**Q5: 如何计算置信区间？**> 根据中心极限定理，样本均值近似正态分布。95%置信区间为：x̄ ± 1.96 × (σ/√n)。当σ未知时，用样本标准差s估计，使用t分布。
