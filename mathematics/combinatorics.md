# 组合数学 (Combinatorics)

## 简介

**组合数学 (Combinatorics)** 是数学的一个分支，研究离散对象的计数、排列、组合以及存在性问题。在计算机科学中，组合数学是算法分析、密码学、概率计算等领域的理论基础。

## 基本计数原理

### 加法原理

```
加法原理：

如果完成一件事有 n 类办法，在第 i 类办法中有 m_i 种不同的方法，
则完成这件事共有：

N = m₁ + m₂ + ... + m_n

示例：从北京到上海
- 坐飞机：3 种选择
- 坐高铁：5 种选择
- 坐汽车：2 种选择

总选择数 = 3 + 5 + 2 = 10

集合表示：
|A ∪ B| = |A| + |B|  (当 A ∩ B = ∅)
```

### 乘法原理

```
乘法原理：

如果完成一件事需要 n 个步骤，第 i 步有 m_i 种方法，
则完成这件事共有：

N = m₁ × m₂ × ... × m_n

示例：组成3位密码
- 第1位：10 种选择 (0-9)
- 第2位：10 种选择
- 第3位：10 种选择

总密码数 = 10 × 10 × 10 = 1000

集合表示：
|A × B| = |A| × |B|
```

## 排列 (Permutations)

### 全排列

```
从 n 个不同元素中取出 n 个元素的排列数：

P(n, n) = n! = n × (n-1) × (n-2) × ... × 2 × 1

示例：3个人排成一队
3! = 3 × 2 × 1 = 6 种排法

排列列表：
ABC, ACB, BAC, BCA, CAB, CBA

复杂度：O(n!)
```

### 选排列

```
从 n 个不同元素中取出 k 个元素的排列数：

P(n, k) = n! / (n-k)!
       = n × (n-1) × ... × (n-k+1)

示例：从5个人中选3个人排队
P(5, 3) = 5! / (5-3)! = 5! / 2! = 5 × 4 × 3 = 60

Python实现：
from itertools import permutations

# 生成所有排列
items = ['A', 'B', 'C', 'D', 'E']
perms = list(permutations(items, 3))
print(len(perms))  # 60
```

### 有重复元素的排列

```
n 个元素中有重复，第 i 类元素有 n_i 个，则排列数：

n! / (n₁! × n₂! × ... × n_k!)

示例：单词 "BANANA" 的排列
- B: 1个
- A: 3个
- N: 2个

排列数 = 6! / (1! × 3! × 2!) = 720 / (1 × 6 × 2) = 60
```

## 组合 (Combinations)

### 基本组合

```
从 n 个不同元素中取出 k 个元素的组合数：

C(n, k) = n! / (k! × (n-k)!)

       = P(n, k) / k!

性质：
- C(n, k) = C(n, n-k)
- C(n, 0) = C(n, n) = 1
- C(n, k) = C(n-1, k-1) + C(n-1, k)  (帕斯卡法则)

示例：从5个人中选3个人组成委员会
C(5, 3) = 5! / (3! × 2!) = 120 / (6 × 2) = 10

Python实现：
from itertools import combinations
from math import comb

# 计算组合数
print(comb(5, 3))  # 10

# 生成所有组合
items = ['A', 'B', 'C', 'D', 'E']
combs = list(combinations(items, 3))
print(len(combs))  # 10
```

### 帕斯卡三角形

```
帕斯卡三角形（杨辉三角）：

        1
       1 1
      1 2 1
     1 3 3 1
    1 4 6 4 1
   1 5 10 10 5 1
  1 6 15 20 15 6 1

每个数等于上方两数之和：C(n,k) = C(n-1,k-1) + C(n-1,k)

性质：
- 第n行和 = 2^n
- 每行对称
- 二项式系数
```

### 可重复组合

```
从 n 个不同元素中可重复地取 k 个元素的组合数：

C(n+k-1, k) = C(n+k-1, n-1)

示例：买3个水果，有苹果、香蕉、橙子3种选择
（可以重复买同一种）

C(3+3-1, 3) = C(5, 3) = 10

可能的组合：
(3,0,0), (2,1,0), (2,0,1), (1,2,0), (1,1,1), 
(1,0,2), (0,3,0), (0,2,1), (0,1,2), (0,0,3)
```

## 二项式定理

```
二项式定理：

(a + b)^n = Σ C(n, k) × a^(n-k) × b^k
          k=0 to n

展开：
(a + b)^n = C(n,0)a^n + C(n,1)a^(n-1)b + ... + C(n,n)b^n

示例：
(a + b)³ = C(3,0)a³ + C(3,1)a²b + C(3,2)ab² + C(3,3)b³
        = a³ + 3a²b + 3ab² + b³

特殊值：
- 令 a = b = 1:  C(n,0) + C(n,1) + ... + C(n,n) = 2^n
- 令 a = 1, b = -1: 交替和 = 0
```

## 容斥原理

```
容斥原理：

对于有限集合 A₁, A₂, ..., Aₙ：

|A₁ ∪ A₂ ∪ ... ∪ Aₙ| = 
  Σ|A_i| 
  - Σ|A_i ∩ A_j| 
  + Σ|A_i ∩ A_j ∩ A_k|
  - ...
  + (-1)^(n+1) |A₁ ∩ A₂ ∩ ... ∩ Aₙ|

两集合：|A ∪ B| = |A| + |B| - |A ∩ B|

三集合：|A ∪ B ∪ C| = |A| + |B| + |C| 
                    - |A∩B| - |B∩C| - |A∩C|
                    + |A∩B∩C|

示例：1-100中能被2或3整除的数有多少？
|A₂| = 50, |A₃| = 33, |A₆| = 16
|A₂ ∪ A₃| = 50 + 33 - 16 = 67
```

## 鸽巢原理

```
鸽巢原理（抽屉原理）：

基本形式：
将 n+1 个物体放入 n 个盒子中，至少有一个盒子包含不少于2个物体。

一般形式：
将 m 个物体放入 n 个盒子中，至少有一个盒子包含至少 ⌈m/n⌉ 个物体。

示例：
1. 13个人中至少有2个人生日在同一个月
   （12个月，13个人）

2. 任意5个整数中，必有3个数的和能被3整除
   （余数只有0,1,2三种）

应用：证明存在性
```

## 卡特兰数 (Catalan Numbers)

```
卡特兰数：

C_n = (1/(n+1)) × C(2n, n)
    = C(2n, n) - C(2n, n+1)

递推：C_0 = 1, C_{n+1} = Σ C_i × C_{n-i}  (i=0 to n)

前若干项：1, 1, 2, 5, 14, 42, 132, 429, 1430, ...

应用场景：
1. 合法括号序列数
2. 二叉搜索树个数
3. 不交叉弦分割
4. 栈排序排列数
5. 网格路径不跨越对角线

示例：n对括号的合法序列数
n=3时，有5种：
((())), (()()), (())(), ()(()), ()()()

Python实现：
def catalan(n):
    from math import comb
    return comb(2*n, n) // (n + 1)
```

## 斯特林数 (Stirling Numbers)

### 第一类斯特林数

```
[s(n, k)]：将 n 个元素分成 k 个非空循环排列的方式数

递推：
s(n, k) = s(n-1, k-1) + (n-1) × s(n-1, k)

边界：s(0, 0) = 1, s(n, 0) = s(0, n) = 0 (n>0)

应用：排列的循环分解
```

### 第二类斯特林数

```
S(n, k)：将 n 个不同元素划分成 k 个非空子集的方式数

递推：
S(n, k) = S(n-1, k-1) + k × S(n-1, k)
（第n个元素单独成集 或 加入已有k个集之一）

边界：S(0, 0) = 1, S(n, 0) = S(0, n) = 0 (n>0)

贝尔数：B_n = Σ S(n, k)  (k=0 to n)
表示n个元素的集合划分数

示例：
S(3, 2) = 3
划分方式：{{1,2},{3}}, {{1,3},{2}}, {{2,3},{1}}
```

## 生成函数

### 普通生成函数

```
序列 {a₀, a₁, a₂, ...} 的生成函数：

G(x) = a₀ + a₁x + a₂x² + a₃x³ + ...

常见序列的生成函数：
1. 常数列 {1,1,1,...}:  G(x) = 1/(1-x)
2. 等差数列 {0,1,2,3,...}: G(x) = x/(1-x)²
3. 几何数列 {1,2,4,8,...}: G(x) = 1/(1-2x)
4. 二项式系数: (1+x)^n

应用：求解递推关系
```

### 指数生成函数

```
序列 {a₀, a₁, a₂, ...} 的指数生成函数：

E(x) = a₀ + a₁x/1! + a₂x²/2! + a₃x³/3! + ...

用于处理排列问题：
- n个元素的排列数：e^x
- 有限制的排列问题

示例：错位排列（derangement）
D_n = n! × Σ(-1)^k/k!  (k=0 to n)
生成函数：D(x) = e^(-x)/(1-x)
```

## 递推关系

### 线性递推

```
斐波那契数列：
F(n) = F(n-1) + F(n-2)
F(0) = 0, F(1) = 1

通项公式（比内公式）：
F(n) = (φ^n - ψ^n) / √5

其中：φ = (1+√5)/2 ≈ 1.618 (黄金比例)
      ψ = (1-√5)/2 ≈ -0.618

快速计算（矩阵快速幂）：
|F(n+1)|   |1 1|   |F(n)  |
|F(n)  | = |1 0| × |F(n-1)|

时间复杂度：O(log n)
```

```python
def fibonacci_matrix(n):
    """矩阵快速幂计算斐波那契数"""
    if n <= 1:
        return n
    
    def multiply(a, b):
        """矩阵乘法"""
        c = [[0, 0], [0, 0]]
        for i in range(2):
            for j in range(2):
                for k in range(2):
                    c[i][j] += a[i][k] * b[k][j]
        return c
    
    def matrix_pow(mat, power):
        """矩阵快速幂"""
        if power == 1:
            return mat
        if power % 2 == 0:
            half = matrix_pow(mat, power // 2)
            return multiply(half, half)
        else:
            return multiply(mat, matrix_pow(mat, power - 1))
    
    mat = [[1, 1], [1, 0]]
    result = matrix_pow(mat, n)
    return result[0][1]
```

## 算法应用

### 组合计数在算法中的应用

```python
# 1. 计算路径数（网格中的路径）
def grid_paths(m, n):
    """从(0,0)到(m-1,n-1)只能向右或向下的路径数"""
    from math import comb
    # 需要 m-1 次向下，n-1 次向右，共 (m+n-2) 步
    return comb(m + n - 2, m - 1)

# 2. 子集枚举
from itertools import combinations

def all_subsets(s):
    """生成所有子集"""
    result = []
    n = len(s)
    for k in range(n + 1):
        for subset in combinations(s, k):
            result.append(subset)
    return result

# 3. 卡特兰数应用：出栈序列数
def count_stack_sequences(n):
    """n个元素的合法出栈序列数"""
    from math import comb
    return comb(2 * n, n) // (n + 1)
```

## 面试要点

### 常见问题

**Q1: C(n,k)和P(n,k)的关系？**
> P(n,k)是从n个元素中有序选取k个，C(n,k)是无序选取。P(n,k) = C(n,k) × k!，即组合数乘以k个元素的全排列。

**Q2: 容斥原理的应用场景？**> 当直接计算并集大小困难，但交集和单个集合大小容易计算时使用。常见于"至少满足一个条件"的计数问题。

**Q3: 卡特兰数的典型应用？**> 1) 合法括号序列 2) 二叉搜索树个数 3) 不交叉弦分割圆 4) 栈的合法输出序列 5) 网格路径不跨越对角线。

**Q4: 如何快速计算大数组合数模p？**> 使用Lucas定理或预处理阶乘和逆元。当n和p较大时，Lucas定理将问题分解为p进制下的子问题。

**Q5: 鸽巢原理的扩展应用？**> 不仅用于证明存在性，还可证明必有某性质满足。如：任意n+1个整数中必有两个数之差能被n整除（余数只有n种）。
